 # Fast Ecommerce 

Ecommerce Front End application under construction.
## Technologies
+ Vue Js
+ Yarn
+ Axios

## Install
```
git clone https://github.com/jonatasoli/fast-ecommerce-front
```
```
yarn install
```
```
yarn serve
``` 

## Compiles ans minfieis for production
```
yarn build
```

## Run your init tests
```
yarn test:unit
```

## Linsts and fixes files

```
yarn lint
```
